# Akhila Mario Resume (static)

This is a one-page, side-scrolling resume inspired by Mario-style movement.

## Run locally
Open `index.html` in a browser (Chrome works great).

## Deploy (GitHub Pages)
1. Create a GitHub repo (e.g., `mario-resume`)
2. Upload everything in this folder to the repo root
3. GitHub → Settings → Pages → Deploy from branch (`main` / root)
4. Add your custom domain in Pages settings and enable HTTPS
